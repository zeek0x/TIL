# ソート

## 安定ソート

安定ソート（stable sort）は、同等なデータのソート前の順序が、ソート後も保存されるものをいう。

例１：学生番号順　成績データ

|学生番号|生徒名|テスト成績|
|---|---|---|
|010|A子|419|
|011|B男|366|
|012|C美|402|
|013|D生|453|
|014|E上|419|
|015|F崎|402|

例２：成績順　安定ソート

|学生番号|生徒名|テスト成績|
|---|---|---|
|011|B男|366|
|012|C美|402|
|015|F崎|402|
|010|A子|419|
|014|E上|419|
|013|D生|453|
